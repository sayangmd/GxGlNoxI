# 项目简介

本系统名为“80-基于深度学习的水面船舶检测识别系统”，主要用途在于通过深度学习技术对水面船舶进行高效检测与识别。系统功能模块包括船舶检测、识别以及用户界面（UI）。技术特点方面，系统采用了yolov5算法，基于Python编程语言开发，并配备有专门的数据集进行模型训练。该系统不涉及售卖或交易信息，专注于技术实现与应用。


## 核心功能
- 船舶检测：通过深度学习技术对水面船舶进行高效检测
- 船舶识别：对检测到的船舶进行分类识别
- 用户界面(UI)：提供图形化操作界面，方便用户使用系统

## 适用场景
- 水面监控：适用于湖泊、河流、海洋等水域的船舶监控
- 安全管理：可用于港口、航道的安全管理
- 环境监测：辅助水域环境监测中的船舶活动分析
- 科研应用：为水域交通研究提供数据支持

## 技术特点
- 基于YOLOv5算法实现高效检测与识别
- 采用Python编程语言开发，便于维护与扩展
- 配备专门数据集进行模型训练，提高识别准确率
- 提供完整的UI界面，操作简便直观
